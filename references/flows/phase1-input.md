# 第一阶段：三项核心输入

> **设计理念**：用户只需提供方向（开篇背景、故事结局、卷纲），Agent根据这三项核心输入自主黑箱生成全部详细设定。无需用户逐一确认世界观、人物、时间线等细节。

---

## 输入一：开篇背景

使用 `AskUserQuestion` 工具询问：

```
Question: 故事的开篇背景是什么？（世界、时代、初始情境）
Options:
- 悬疑推理（当代都市/封闭空间/连环案件）
- 言情（现代都市/古代宫廷/穿越时空）
- 奇幻玄幻（异世界/修真大陆/魔法文明）
- 科幻未来（星际/末世/赛博朋克）
- 武侠仙侠（江湖/门派/修仙界）
- 历史架空（特定朝代/虚构王朝/权谋朝堂）
- 自由描述（我有完整的世界构想）
```

**记录**：`开篇背景 = [用户选择/描述]`

如用户选择自由描述，完整记录其世界设定、时代背景、初始情境、核心矛盾等。

---

## 输入二：故事结局

使用 `AskUserQuestion` 工具询问：

```
Question: 故事最终走向和结局是怎样的？
Options:
- 大团圆结局（主角成功，众人归位）
- 悲剧结局（主角牺牲/遗憾收场）
- 开放式结局（留有余韵和想象空间）
- 反转结局（出人意料的真相/逆转）
- 苦涩胜利（代价沉重的胜利）
- 自由描述（我已想好结局）
```

**记录**：`故事结局 = [用户选择/描述]`

追问（可选）：
```
Question: 结局时主角的状态是？（可跳过，Agent自行安排）
Options:
- 登顶巅峰/功成名就
- 归隐田园/平淡幸福
- 牺牲自我/成全他人
- 看破红尘/超然物外
- 留下遗憾/未完待续
- 自由描述
```

---

## 输入三：卷纲

使用 `AskUserQuestion` 工具询问：

```
Question: 计划分几卷/几章？各卷的大体内容和结尾效果是？
Options:
- 10章短篇（约3-5万字，1卷）
- 20章中篇（约6-10万字，2-3卷）
- 30章中长篇（约9-15万字，3-4卷）
- 50章长篇（约15-25万字，5-6卷）
- 自定义章节数和卷数
```

**记录**：`总章数 = [用户选择]`

追问卷纲详情：
```
Question: 请描述各卷的大体内容和卷末效果（简要即可，Agent会自动细化）
示例格式：
  第一卷（第1-10章）：主角初入江湖，卷末发现身世秘密
  第二卷（第11-20章）：拜师学艺，卷末与宿敌首次对决
  第三卷（第21-30章）：终极对决，卷末大结局
请自由描述你的卷纲：
```

**记录**：`卷纲 = [用户描述的各卷内容和卷末效果]`

---

## 输入完成 → 展示确认

三项输入全部完成后，展示摘要：

```markdown
## 核心输入确认

🌍 **开篇背景**：[用户输入的背景描述]
🎬 **故事结局**：[用户输入的结局描述]
📚 **卷纲规划**：
  [用户输入的各卷内容和卷末效果]
```

使用 `AskUserQuestion` 请求确认：

```
Question: 以上是你的核心创作方向，确认后我将自动生成全部设定（世界观、人物卡、大纲体系等），无需你逐项确认细节。
Options:
- 确认，开始自动生成！
- 我想修改某些内容
- 我想补充更多细节
```

用户确认后 → 进入第二阶段：黑箱生成全部设定。
