# 网文叙事技法精要

> 融合番茄/起点/晋江头部作者实战技法，覆盖感官沉浸、信息释放、情绪操控、视角魔术四大维度。
> 本文件为 Phase 3 每章创作时的核心参考，与 chapter-guide.md（结构）、hook-techniques.md（钩子）、dialogue-writing.md（对话）配合使用。

## 一、感官沉浸系统

### 核心法则：五感轮转，禁止单感独白

任何场景描写必须覆盖至少三种感官，且不能连续两句用同一感官。

**感官优先级（网文适配）：**
视觉 > 听觉 > 触觉 > 嗅觉 > 味觉

**实操节奏：** 视觉打底→听觉推进→触觉/嗅觉点睛→味觉收束（仅限食戏/亲密场景）

### 感官替换词库（禁止通用化）

| 感官 | 弱写法（AI常见） | 强写法（真人作者） |
|------|-----------------|-------------------|
| 视觉 | 美丽的/明亮的/昏暗的 | 用具体光源+色温：日光灯惨白/烛火把影子拉成歪脖子树 |
| 听觉 | 吵闹的/安静的/悦耳的 | 用具体声源+质感：铁锅铲刮锅底的尖锐声/远处火车经过的闷雷声 |
| 触觉 | 温暖的/冰冷的/粗糙的 | 用具体材质+反应：铁栏杆的凉顺着掌心往上爬/风把耳朵刮得像刀片 |
| 嗅觉 | 芳香的/刺鼻的/清新的 | 用具体气味源+联想：医院走廊的消毒水味混着方便面调料包的咸香 |

### 环境情绪映射三法

**1. 同向放大：** 环境强化角色当前情绪
> 他走在雨里。雨点砸在伞面上，像有人在用力鼓掌，庆祝他今天的狼狈。

**2. 反向撕裂：** 环境与角色情绪形成反差，制造张力
> 外面放着烟花，五彩斑斓炸了满天。她坐在出租屋地板上，把结婚照撕成四片。

**3. 隐喻渗透：** 环境暗示即将到来的命运
> 楼道的灯泡忽明忽暗，她每走一步，影子就分裂成两个，又合拢成一个。

## 二、信息释放的节奏控制

### 核心法则：永远不要一次倒完

读者的阅读快感来源于"知道一点、想知道更多"。信息释放遵循「七分饱」原则——每次只给读者70%的答案，留30%的悬念驱动下一章。

### 五层信息释放梯度

| 层级 | 策略 | 效果 | 适用场景 |
|------|------|------|---------|
| L1 暗示 | 不直接说，用细节暗示 | 读者隐约觉得不对 | 伏笔埋设 |
| L2 碎片 | 只透露信息的一小块 | 读者开始拼凑 | 真相铺垫 |
| L3 半遮 | 说一半留一半 | 读者急切想知道另一半 | 中期悬念 |
| L4 揭底 | 完整揭示真相 | 读者恍然大悟 | 高潮反转 |
| L5 颠覆 | 揭示的真相推翻之前的认知 | 读者震惊 | 核心反转 |

**示例——一个身世秘密的信息释放：**
- L1（第2章）：主角摸到后颈有一道旧疤，但记不清怎么来的。
- L2（第5章）：老照片里，婴儿的襁褓上绣着一个他不认识的族徽。
- L3（第10章）：临死的反派说："你以为你姓张？你该姓沈。"
- L4（第18章）：DNA报告证实，他确实是沈家流落在外的血脉。
- L5（第25章）：但沈家当年抛弃他，是因为他身上携带着一种被禁止的基因。

### 信息炸弹放置时机

信息炸弹 = 在读者以为安全的时候引爆意外信息。

**最佳放置位置：**
1. 日常场景的结尾（温馨→突变）
2. 角色放下戒备的瞬间（安全→危机）
3. 读者以为真相大白的下一秒（答案→新的疑问）
4. 章节最后三行（最大化钩子效果）

## 三、情绪操控的四个开关

### 开关一：共鸣锚点

让读者在角色身上看到自己。不需要角色经历和读者一样，只需要情绪底层是共通的。

**共鸣四锚：**
- **不公平感**：明明努力了，为什么还是不行？
- **被误解**：我知道真相，但没人信我。
- **孤独感**：身边都是人，但没人真正懂我。
- **不甘心**：我差一点就做到了。

**操作：** 每三章至少植入一个共鸣锚点。在角色遭遇重大挫折时，用内心独白或小动作把锚点落地。

### 开关二：情绪温差

连续高强度情绪会让读者麻木。必须在高情绪段落之间插入低情绪段落（日常、幽默、温馨），制造温差。

**节奏公式：**
- 3段压抑 → 1段喘息 → 2段压抑 → 1段爆发
- 每个情绪波峰后，必须有至少200字的情绪缓冲
- 大虐之后必有小甜（哪怕是角色间的默契一笑）

### 开关三：延迟满足

读者最想看到的场景，不要马上给。先让读者等一等，等到他们快受不了的时候再给，快感翻倍。

**延迟三板斧：**
1. **即将达成时被打断**：手刚碰到门把手，电话响了。
2. **虚假满足后收回**：他以为赢了，才发现中了圈套。
3. **旁观者视角延迟**：不直接写主角的反应，先写旁人的表情。

### 开关四：细节情感化

同样是写"她哭了"，细节不同，情绪强度天差地别。

| 弱写法 | 强写法 |
|--------|--------|
| 她哭了 | 她把脸埋进围巾里，肩膀一抖一抖的，鼻涕和眼泪糊在一起，狼狈得不像话 |
| 他很伤心 | 他蹲在路边，点了三次打火机都没点着烟，最后把烟捏断了 |
| 她笑了 | 她笑得太大声了，笑到最后声音变了调，像是喉咙里卡了什么东西 |

## 四、视角魔术

### 核心法则：限知视角是最大的悬念制造机

第三人称限知视角 = 读者只知道主角知道的。这意味着：
- 主角被蒙骗时，读者也被蒙骗（直到真相揭晓的爽感）
- 主角看不到的地方发生的事，读者也看不到（制造悬念）
- 主角的判断可能是错的（支持反转）

### 视角越界的红线

| 越界类型 | 示例 | 问题 |
|---------|------|------|
| 突然跳到反派视角 | "与此同时，反派在密室里笑了" | 破坏限知，削弱悬念 |
| 主角不在场却描写现场 | "房间里，凶手把刀藏在了床下" | 读者知道了主角不该知道的事 |
| 全知评论 | "他不知道，这将是他最后一次笑" | 跳出叙事身份，破坏沉浸感 |

**正确处理：** 如果必须展示主角不在场的信息，通过以下方式：
- 其他角色转述（不可靠叙述，增加悬念）
- 主角事后发现线索（推理链条）
- 信件/录音/监控等道具（符合世界观逻辑）

### 多视角切换规范（如适用）

- 每次切换视角必须有明确的场景分割标记
- 同一章内最多切换2次视角
- 切换时必须在前一个视角的冲突高点切入（读者急着看后续，但被迫看另一个视角 = 悬念）
- 每个视角角色必须有独立的叙事声音（措辞、思维模式、关注点不同）
- 上帝视角叙述仅限于全书开头的第一段（建立世界观基调），之后锁定限知

## 五、网文特有的节奏技法

### 黄金三章法则

网文读者的耐心比出版物读者更短。前三章（约9000-15000字）必须完成：
1. **第1章**：建立主角身份 + 核心冲突点燃 + 第一个钩子
2. **第2章**：金手指/核心能力初显 + 读者预期建立 + 第二个钩子
3. **第3章**：第一个小高潮 + 主角性格全面展示 + 读者决定追读

### 每章节奏曲线（网文适配）

`
张力
 ^
 |        /\          /\
 |       /  \    /\  /  \      <- 至少2个波峰
 |      /    \  /  \/    \
 |     /      \/          \   <- 波谷=喘息+铺垫
 |    /                    \
 +---+----+----+----+----+-----> 字数
    0   1k   2k   3k   4k   5k
`

**硬性规则：**
- 开头500字内必须出现第一个冲突/悬念
- 连续500字无冲突 → 必须插入新张力
- 结尾200字内必须设置钩子
- 全章至少2个张力波峰

### 爽点/虐点配比

| 题材 | 爽:虐 比例 | 说明 |
|------|-----------|------|
| 爽文/升级流 | 7:3 | 七分爽三分虐，虐是为了更爽 |
| 言情/甜宠 | 6:4 | 甜为主，刀为辅 |
| 悬疑/推理 | 5:5 | 信息揭露的爽感 + 真相的残酷 |
| 虐文/致郁系 | 3:7 | 虐为主，偶尔给甜让读者缓口气 |

## 六、段落呼吸感

### 网文段落黄金长度

| 内容类型 | 推荐段落长度 | 原因 |
|---------|------------|------|
| 对话交锋 | 1-2句/段 | 快节奏，视觉轻 |
| 动作场面 | 2-4句/段 | 模拟镜头切换速度 |
| 心理活动 | 3-5句/段 | 给读者沉浸的空间 |
| 环境描写 | 4-6句/段 | 建立画面感但不拖沓 |
| 信息交代 | 5-8句/段 | 必须精炼，超过就拆 |

### 长短句交替实操

禁止连续3句以上长度相近的句子。必须有意识地交替：

`
短：他停住了。
长：走廊尽头的灯管发出滋滋的电流声，忽明忽暗地照着地面上那摊还没干透的深色液体。
短：是血。
长：他蹲下来，用手指沾了一点凑到鼻子前——不是铁锈味，是油漆。红色的油漆。
短：谁在这里刷油漆？
`

### 对话与叙述的穿插节奏

纯对话流（连续5句以上无动作/心理/环境描写）是AI通病。正确节奏：

`
对话 → 动作/微表情 → 对话 → 环境/感官 → 对话 → 心理活动 → 对话
`

每次对话之间至少插入一句非对话内容（可以是一个动作、一个眼神、一个环境细节）。
