# 人物塑造指南

好的人物是故事的灵魂。读者记住的是人，不是情节。

> 人物档案模板请使用 [character-template.md](character-template.md)（Phase 2 创建时使用）。本文档聚焦于**在写作过程中**塑造人物的技法。

---

## 核心原则：矛盾创造深度

真实的人是复杂的，充满矛盾。矛盾不是缺陷，而是人物的**发动机**。

**有效矛盾示例：**
- 雷厉风行的女警长，回家后对着亡夫的照片说话
- 满口仁义的书生，暗中替人写休书赚银子
- 冷血杀手，每天给流浪猫带鱼
- 最不信命的人，抽屉里藏着一张求来的护身符

**无效矛盾**：为矛盾而矛盾，与人物核心无关。

## 侧面揭示技法

不要直接陈述性格，通过行为展示。这是"展示而非讲述"在人物塑造上的核心应用。

### 用动作替代标签

| 直接陈述（弱） | 侧面展示（强） |
|----------------|----------------|
| 他很愤怒 | 他捏碎手中的纸杯，碎片扎进掌心也没松手 |
| 她很紧张 | 她反复调整眼镜位置，第三次了 |
| 他很傲慢 | 他从不直视下属的眼睛，对话时总是看窗外 |
| 她很善良 | 她偷偷喂了三年流浪狗，谁也没告诉 |
| 他很孤独 | 电视永远开着，他从来不看 |

### 用选择揭示本质

人物在压力下的选择，比任何描述都有说服力。

**示例：**
> 火灾现场，他可以救名画，也可以救那只猫。他抱起了猫。
> 不需要写"他善良"，这个选择已经说了。

### 用习惯暴露真相

反复出现的小习惯是人物的无意识自我暴露。

- 紧张时摸无名指（曾经戴过戒指？）
- 总是坐在背靠墙的位置（缺乏安全感？）
- 说话时避免眼神接触（隐瞒什么？）
- 对每个服务员都说谢谢（教养？还是内疚？）

## 五类角色分层创作规范

本技能使用**五类角色分层体系**，严格区分每类角色的创作维度，杜绝脸谱化、同质化。详见下文。

---

### ① 主角 (Protagonist)

#### 核心要素
- **明确目标** - 他想要什么
- **强大动机** - 为什么想要
- **可共情性** - 读者能理解他的感受
- **成长空间** - 故事中会改变

#### 三维人设模板
- **原生性格**：故事开始时的性格状态
- **成长性格蜕变线**：从起始状态到终局的性格变化路径
- **身世秘密**：推动角色行为的深层背景
- **核心执念**：驱动一切行动的底层信念
- **致命软肋**：会被对手利用的真正弱点
- **专属优缺点**：差异化特质组合
- **心境变化周期**：情绪在不同阶段的波动规律
- **完整人物弧光**：起-承-转-合的完整轨迹
- **成长代价**：每一次突破实际付出了什么
- **最终宿命**：故事的终点

#### 主角原型

| 类型 | 特征 | 故事作用 |
|-----|------|---------|
| 英雄型 | 勇敢、正义、利他 | 战胜外在威胁 |
| 成长型 | 从弱小到强大 | 克服内在缺陷 |
| 反英雄型 | 道德灰色、复杂 | 挑战传统道德 |
| 平凡型 | 普通人卷入非凡事 | 读者代入感强 |

#### 缺陷致命化

主角必须有缺陷，缺陷在关键时刻导致失败。没有缺陷的角色不会成长。

| 缺陷 | 导致的失败 | 成长方向 |
|-----|----------|---------|
| 傲慢 | 低估对手，落入陷阱 | 学会倾听 |
| 信任问题 | 拒绝帮助，孤立无援 | 学会信任 |
| 完美主义 | 无法及时行动，错失机会 | 学会接受不完美 |
| 复仇心 | 被利用，失去理智 | 学会放下 |
| 逃避 | 问题越滚越大 | 学会面对 |

#### 行动逻辑原则
所有行动有底层逻辑支撑，不圣母、不降智、不OOC，成长循序渐进，有高光也有缺憾。行为必须可追溯、可理解、符合角色当前状态。

### ② 重要角色（核心配角/羁绊角色）

#### 功能性关系定义

| 类型 | 核心特征 | 写作要点 | 示例 |
|-----|---------|---------|------|
| 导师型 | 指引主角，传递信息 | 必须有自己的盲区 | 师父、前辈 |
| 盟友型 | 协助主角，提供支持 | 不能只是工具人 | 伙伴、战友 |
| 镜像型 | 与主角共鸣或对照 | 展示主角的可能性 | 知己、对手 |
| 催化剂型 | 推动主角行动 | 出场时机精准 | 事件触发者 |
| 阻碍型 | 制造内部冲突 | 并非反派，立场不同 | 保守派、竞争者 |

#### 创作标准
- 拥有**独立支线**：和主线有关联但不完全依附
- **个人执念与软肋**：有自己的渴望和恐惧
- **与主角强羁绊**：亲情/友情/爱情/师徒情
- **专属剧情高光与遗憾**：有独立的高光时刻和缺憾
- 人设差异化极强，是主线剧情、刀子伏笔、主角成长的核心推动者
- 拥有**独立结局闭环**，角色弧线完整

#### 独立性检查清单
- [ ] 删除这个角色，主线是否受影响？
- [ ] 他有自己的欲望和恐惧吗？
- [ ] 他的行动动机是否独立于主角？
- [ ] 他有自己的故事线吗（即使很简短）？

### ③ 次要角色（功能性配角）

#### 快速构建三要素
1. **鲜明特征**：一个让人记住的点（外貌、口癖、习惯动作）
2. **明确功能**：为谁服务、达到什么效果
3. **微小动机**：他为什么在做这件事

#### 功能性配角分类

| 类型 | 功能 | 注意 |
|-----|------|------|
| 信息提供者 | 传递关键线索 | 方式要自然，不像是教科书 |
| 场景营造者 | 增加世界真实感 | 有鲜明的身份标记 |
| 冲突制造者 | 引发小规模对立 | 动机要合理 |
| 情感触发者 | 引发主角情绪变化 | 存在感适度 |

#### 出场与退场原则
- 不冗余、不工具人
- 出场合理、退场自然
- 可铺垫伏笔、推动细节剧情
- 避免工具人化的技巧：

| 工具人写法 | 立体化写法 |
|-----------|-----------|
| "服务员端来了咖啡" | "服务员是个五十来岁的女人，手上戴着两个银戒指，放下咖啡时看了他一眼，欲言又止。" |
| "出租车司机说" | "司机是个话痨，从天气聊到房价，但他没有回应。司机从后视镜里看了他一眼，识趣地闭嘴了。" |

### ④ 重要反派（核心BOSS/阶段性强敌）

#### 核心标准
好反派不是坏人，是**信念不同的人**。

- **强大可信** - 不应该是草台班子
- **有自己的逻辑** - 他相信自己在做正确的事
- **与主角有深层联系** - 不是单纯为了作恶
- **揭示主题** - 挑战主角的信念
- **完整反派弧光** - 从出场到陨落有完整成长线与陨落逻辑

#### 三层动机模型

**浅层动机**（不够）：
> "我想毁灭世界" -> 乏味

**中层动机**（及格）：
> "我失去了一切，世界对我没有意义" -> 可理解但单薄

**深层动机**（有力）：
> "我的女儿死于一种本可治愈的病，但药厂为了利润延迟了上市。这个世界不值得被拯救。" -> 读者虽然不认同，但能理解

#### 反派与主角的镜像关系

最好的反派是主角的**黑暗镜像**——如果主角走了另一条路，就会变成他。

```
主角：失去家人后选择保护他人
反派：失去家人后选择毁灭一切
同一起点，不同选择。
```

#### 反派塑造标准

1. 拥有**独立身世**：有完整背景故事
2. 有**无奈与执念**：非纯粹恶人设
3. 有**信仰与底层逻辑**：其行为有自洽的理由
4. 有**个人魅力**：读者能理解甚至同情
5. 与主角形成**宿命对立**：冲突对立有理有据

#### 出场与退场原则

- 重要反派逐步登场，前期留有暗示
- 阶段性退场（暂退而非死亡）可以有
- 最终退场必须符合人物逻辑和剧情需要
- 退场时应有遗留影响

---

### ⑤ 次要反派（阶段性敌对势力/小人物反派）

#### 创作原则
- 定位清晰、动机明确
- 因利益、立场、偏见、阵营对立产生冲突
- 行为逻辑自洽
- 用于铺垫阶段性矛盾、衬托主线冲突、推动剧情过渡

#### 类型

| 类型 | 特征 | 作用 |
|-----|------|------|
| 炮灰型 | 为主角展示实力 | 快速打脸 |
| 立场型 | 不同阵营/利益冲突 | 展示世界观 |
| 执念型 | 对主角有私人恩怨 | 制造阶段性危机 |
| 误会型 | 因误解产生冲突 | 铺垫反转 |

#### 创作要点
- 不需要完整背景故事，但有清晰动机
- 不需要强大实力，但在其领域有优势
- 可以转化（由敌变友），但要合理铺垫
- 退场方式不重要，但消失不能太突兀

#### 五类角色关系总览

```
       重要反派 (宿命对立)
        ↑         ↓
主角 ←──────────→ 重要角色 (羁绊连接)
        ↑         ↓
       次要反派 (阶段性冲突)    次要角色 (功能性支持)
```

## 人物关系设计

### 关系动态变化

好的关系会随故事发展，不是静态的：

```
第一章：陌生人
第三章：不情愿的盟友
第五章：建立信任
第七章：背叛/考验
终章：真正的友谊（或决裂）
```

### 关系揭示层次

逐步揭示关系深度，不要一次性全部暴露：
- **表层**：表面互动
- **中层**：共同经历后的默契
- **深层**：真实感受/秘密

### 高戏剧潜力关系

| 关系类型 | 戏剧潜力 | 经典模式 |
|---------|---------|---------|
| 亦敌亦友 | 高 | "我知道你在利用我，但我需要你" |
| 禁忌之爱 | 高 | 身份/立场对立，但无法控制感情 |
| 师徒关系 | 中 | "教会徒弟，饿死师父"的潜在张力 |
| 兄弟竞争 | 中 | 同一起点，不同选择，此消彼长 |

## 人物出场设计

### 首次出场原则

有效的出场方式：
- **行动中** - 展示能力或性格
- **冲突中** - 立即建立关系/对立
- **误解中** - 建立悬念

### 出场示例对比

| 无效出场 | 有效出场 |
|---------|---------|
| 李明，28岁，是一名侦探。他走进办公室。| 李明跨过警戒线，警官试图拦住他。"市刑警队，李明。"他亮出证件，径直走向尸体。|
| 美丽的女孩坐在窗边，她叫小红。| 她已经三天没睡了，咖啡杯里的液体在颤抖。当门铃响起时，她几乎把杯子摔在地上。|

**避免**：镜子自照描写外貌、简历式介绍、无意义的日常活动。

## 人物一致性检查

角色行为必须符合已建立的性格。

**检查问题：**
- 这件事符合他的核心价值观吗？
- 以他的背景，会有这样的反应吗？
- 他的恐惧会如何影响这个决定？
- 他的缺陷会导致他犯什么错？

**例外处理：**
- 如果角色"不符合性格"行事，必须有原因
- 解释应该在相同/下一章提供
- 可以是成长的标志（角色克服了缺陷）

**五类角色各维度交叉检查：**
- [ ] 主角的成长路径是否需要某个重要角色的牺牲/推动？
- [ ] 重要反派的某个设定是否与主角形成镜像对照？
- [ ] 重要角色的独立支线是否与主线产生了有意义的交叉？
- [ ] 次要角色的出场是否过于频繁或过于工具化？
- [ ] 次要反派的退场是否为主角制造了新的冲突？
